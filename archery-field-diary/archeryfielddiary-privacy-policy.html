개인정보처리방침 개요

습사일지 앱은 개인정보보호법, 정보통신망법 등 관련 법령을 준수하며, 사용자의 개인정보를 보호하기 위해 다음과 같이 개인정보를 처리합니다.

개인정보 수집 및 이용

수집하는 개인정보 항목

  • 앱 사용 통계: 기능 사용 빈도, 세션 시간, 오류 로그
  • 보안 세션 정보: 사용자 ID, 로그인 시간 (보안 모니터링용)
  • 광고 식별자(ADID): 맞춤형 광고 제공을 위한 기기별 식별자
  • 훈련 기록 데이터: 화살 수, 점수, 날짜, 위치, 훈련 내용 등 사용자가 입력한 궁도 훈련 정보 (백업 기능 이용 시)
  • 기기 정보: OS 버전, 앱 버전, 기기 모델명
  • 위치 정보: Google Mobile Ads를 통한 IP 기반 대략적 위치 정보 (광고 타겟팅 목적)
  • Google 계정 정보: 이메일 주소 (백업 서비스 이용 시)
※ 참고사항: 본 앱은 GPS 위치 권한을 요청하지 않으며, 정확한 GPS 위치 정보는 수집하지 않습니다. 훈련 기록에 포함되는 "위치"는 사용자가 직접 입력한 텍스트 형태의 장소명입니다. Google Mobile Ads SDK는 광고 타겟팅을 위해 IP 기반의 대략적인 위치 정보를 수집할 수 있습니다.

수집 목적

  • 앱 개선: 사용 패턴 분석을 통한 기능 개선 및 버그 수정
  • 보안 모니터링: 의심스러운 활동 감지 및 계정 보안 강화
  • 맞춤형 광고: 사용자 관심사 기반 광고 제공
  • 데이터 백업: 사용자 요청 시 클라우드 백업 서비스 제공
  • 고객 지원: 문의사항 처리 및 기술적 문제 해결

처리 근거

  • 서비스 제공을 위한 필요 (개인정보보호법 제15조 제1항 제1호)
  • 보안을 위한 필요 (개인정보보호법 제15조 제1항 제1호)
  • 이용자의 동의 (개인정보보호법 제15조 제1항 제2호)
  • 광고 제공을 위한 마케팅 동의 (개인정보보호법 제15조 제1항 제2호)

수집 방법

  • Google Analytics, Firebase를 통한 자동 수집
  • 사용자가 백업 기능을 활성화할 때 수동 수집
  • 앱 내 설정에서 사용자 직접 동의

데이터 저장 및 관리

훈련 기록 데이터

  • 사용자의 훈련 기록은 기본적으로 기기에 로컬로 저장
  • AES-256 암호화를 사용하여 저장
  • 백업 기능 사용 시 Firebase를 통해 클라우드 서버에 암호화되어 저장
  • 앱 삭제 시 기기에 저장된 로컬 데이터는 즉시 삭제됩니다

앱 사용 통계

  • Google Analytics를 통해 수집 및 Google 서버에 저장
  • Google의 데이터 보존 정책에 따라 관리:
    • 광고 식별자: 13개월 후 자동 삭제
    • 사용 통계 데이터: 익명화 처리 후 최대 2년간 보관

보안 세션 정보

  • Firebase Firestore를 통해 저장
  • 보관 기간: 최대 30일 (이후 자동 삭제)
  • 삭제 방법: 앱 내 계정 삭제 시 자동 삭제 또는 30일 경과 후 자동 삭제

제3자 서비스 이용 현황

본 앱은 서비스 개선과 안정적인 운영을 위해 Google의 클라우드 서비스를 이용합니다.

※ 본 앱은 아래에 명시된 Google 서비스 외에는 제3자와 개인정보를 공유하지 않습니다.

Google 서비스 이용

  • 서비스 제공자: Google LLC
  • 이용 목적: 앱 성능 분석, 맞춤형 광고 제공, 데이터 백업 서비스
  • 처리되는 정보: 앱 사용 통계, 광고 식별자, 기기 정보, 훈련 기록 (백업 이용 시)
  • 데이터 보관: Google 개인정보처리방침에 따라 안전하게 관리

Firebase 서비스

  • 제공받는 자: Google LLC
  • 제공 목적: 데이터 백업, 사용자 인증, 앱 성능 모니터링
  • 제공 정보: 훈련 기록 (사용자 동의 시에만), Google 계정 정보, 보안 세션 정보 (사용자 ID, 로그인 시간)
  • 보유 및 이용기간: 계정 삭제 시 모든 데이터가 즉시 삭제되며, 사용자 삭제 요청 시까지 보관됩니다

Google Mobile Ads (AdMob) 서비스

  • 제공받는 자: Google LLC
  • 제공 목적: 맞춤형 광고 제공 및 광고 성과 측정
  • 제공 정보: 광고 식별자(ADID), 기기 정보, 앱 사용 통계, IP 기반 대략적 위치 정보
  • 위치 정보 처리: GPS 위치는 수집하지 않으며, 광고 타겟팅을 위한 IP 기반 대략적인 위치만 수집됩니다
  • 데이터 보관: Google 개인정보처리방침에 따라 관리되며, 광고 식별자는 13개월 후 자동 삭제
  • iOS 사용자: iOS 14.5 이상 버전에서는 App Tracking Transparency를 통해 광고 추적 권한을 요청하며, 사용자가 거부할 경우 광고 식별자 수집이 제한될 수 있습니다

사용자 제어권

사용자는 다음 방법으로 데이터 수집을 제어할 수 있습니다:

  • 앱 내 설정에서 백업 기능 활성화/비활성화
  • Google 계정 설정에서 광고 개인화 설정 변경
  • 기기 설정에서 광고 추적 제한 설정 (iOS: 설정 > 개인정보 보호 > 추적, Android: 설정 > Google > 광고)
  • iOS 사용자는 앱 실행 시 표시되는 추적 권한 요청에서 허용/거부 선택 가능

사용자 권리

  • 열람권: 언제든지 개인정보 처리 현황을 확인할 수 있습니다
  • 정정·삭제권: 저장된 데이터를 수정하거나 삭제할 수 있습니다
  • 처리정지권: 개인정보 처리 중단을 요청할 수 있습니다
  • 광고 ID 재설정: 기기 설정에서 광고 식별자를 재설정할 수 있습니다
  • 권리 행사 방법: 아래 연락처로 언제든 문의 가능

데이터 삭제 요청 방법

  • 앱 내 계정 삭제 기능을 통한 삭제: 설정 > 계정 관리 > 계정 삭제
  • 이메일을 통한 삭제 요청: maxtro7k@gmail.com으로 요청
  • 처리 기간: 삭제 요청 접수 후 7일 이내 처리

보안 조치

  • 로컬 데이터는 암호화를 사용하여 저장
  • Google 서비스는 업계 표준 보안 프로토콜 사용
  • 개인 식별이 가능한 민감정보는 수집하지 않음
  • 정기적인 앱 업데이트를 통해 보안을 강화합니다
